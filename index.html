Technology

We formally verify programs by automatically translating source code to the proof system Coq. We then prove arbitrarily complex properties, either automatically or manually. By integrating into continuous integration systems, we make sure proofs are kept up-to-date with the code.

As an example, our technology is used for the crypto-currency Tezos to formally verify its implementation. We believe Tezos to be the first crypto-currency with an ongoing process of formal verification of its implementation, making it one the safest decentralized platform. See the coq-tezos-of-ocaml project for the current status.

What do we support

Our tool chain supports the following languages:

  • TypeScript
  • OCaml / ReScript
  • Haskell
  • Go

We focus on easy-to-catch but safety critical properties.
